# To begin program:
## Create Enviroment
```
python3 -m venv env
```

## Start Enviroment
```
source env/bin/activate
```

## Install Packages 
```
pip install yfinance
```

## Install Source Code
```
git clone https://github.com/andykr1k/StockDataScraper.git
```

## Run Program
```
python3 Scraper.py {Ticker}
```

## Don't Forget to Deactivate your Enviroment!
```
deactivate
```
